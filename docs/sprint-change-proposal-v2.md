## План реализации (обновлённый)

### Фаза 1: Публичный сайт (недели 1-3)
**Приоритет: МАКСИМАЛЬНЫЙ**

- Главная = интерактивная карта
- Определение геолокации
- Маркеры объектов с ценами
- Фильтры: город, тип, даты, цена, удобства
- Список объектов с миниатюрами
- Страница объекта (галерея, описание, календарь)
- PWA (установка на телефон)

**Результат**: Пользователи могут искать и смотреть объекты

### Фаза 2: Авторизация и бронирование (недели 4-5)

- Регистрация/вход по Email
- JWT токены
- Создание бронирования
- Мои бронирования

**Результат**: Клиенты могут бронировать

### Фаза 3: Кабинет партнёра (недели 6-8)

- Онбординг партнёра (данные компании, реквизиты)
- Создание объектов (мастер)
- Dashboard партнёра
- Мои клиенты
- Автогенерация договоров PDF

**Результат**: Партнёры могут работать на платформе

### Фаза 4: Панель сотрудника (недели 9-10)

- Dashboard с метриками
- Все партнёры/объекты/клиенты/бронирования
- Модерация объектов
- Фильтры по компании

**Результат**: Staff контролирует платформу
