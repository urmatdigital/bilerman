**Дата**: 2026-01-28
**Инициатор**: Урмат

**Статус**: Готов к утверждению

---

## 1. Резюме изменения

### Что произошло
Принято решение о трансформации проекта **KONOK** (платформа бронирования с интеграциями) в **BILERMAN.KG** — локальную платформу безопасного бронирования жилья в Кыргызстане с уникальным УТП эскроу-защиты платежей.

### Триггер изменения
Стратегический пивот на основе бизнес-документации BOOGUARD:
- Бизнес-план 2026-2028
- Техническая архитектура
- GTM стратегия
- Юридическая защита и гарантии
- Партнёрское предложение
- Инвесторская презентация

### Масштаб изменения
**MAJOR** — Требуется переработка бизнес-логики, добавление платёжной системы с эскроу, изменение аутентификации.

## 2. Анализ воздействия

### 2.1 Что сохраняется (70-80%)
| Компонент | Описание |
|-----------|----------|
| Frontend стек | React 19 + Vite + TypeScript + Tailwind |
| Backend стек | FastAPI + Python 3.11 + PostgreSQL |
| UI компоненты | Atomic Design система, Radix UI |
| Карты | Mapbox GL интеграция |
| Кеширование | Redis |
| Хостинг | Vercel + Railway |
| i18n | Мультиязычность (RU, KY) |

### 2.2 Что добавляется
| Компонент | Описание | Сложность |
|-----------|----------|-----------|
| OTP аутентификация | Вход по SMS вместо email | Средняя |
| Платежи (эскроу) | MBank, O!Dengi, Элсом | Высокая |
| Система неустоек | Политики отмены, штрафы | Средняя |
| Верификация | Партнёры и объекты | Средняя |
| Выплаты партнёрам | Автоматические после check-in | Средняя |

### 2.3 Что удаляется
| Компонент | Причина |
|-----------|---------|
| Интеграции Booking.com/Airbnb | Не нужно для локального MVP |
| Модуль клинеров | Не в скоупе BILERMAN |
| Микросервисы коммуникаций | Упрощение до SMS |
| AI чаты (CopilotKit) | Не приоритет |
| Динамическое ценообразование | Не для MVP |

## 3. Рекомендуемый путь

### Выбранный подход: Эволюция существующего кода

**Обоснование:**
1. 70-80% кода переиспользуется
2. Меньше рисков, чем начинать с нуля
3. Быстрее достичь MVP
4. Команда знакома с кодовой базой

### Альтернативы (отклонены):
- ❌ Новый проект на Next.js + Supabase — слишком долго, потеря существующего кода
